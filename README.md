# streamlit App
Wichtig für die Zukunft:
Jedes Mal, wenn du ein neues Terminal öffnest und an deiner App arbeiten willst, musst du zwei Dinge tun:

In den Ordner gehen: 
```bash
cd ~/github/streamlit
```

Die Umgebung aktivieren: 

```bash
source venv/bin/activate
```

wie starte ich eine Streamlit app?

```bash
streamlit run roadbook.py
```
